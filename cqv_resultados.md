# Resultados del Análisis CQV (Quality and Structural Value)

Este documento contiene la evaluación completa del **CQV Score** para la lista de empresas provista, ordenadas de mayor a menor calidad según el modelo matemático y financiero establecido.

## 📊 Ecuación del CQV Score
$$CQV = (F_1 \times 0.25) + (F_2 \times 0.15) + (F_3 \times 0.15) + (F_4 \times 0.25) + (F_5 \times 0.20)$$

Donde:
* **F1**: Rentabilidad (25%)
* **F2**: Solidez Financiera (15%)
* **F3**: Crecimiento Eficiente (15%)
* **F4**: Moat Actual (25%)
* **F5**: Proyección Futura (20%)
